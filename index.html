What RESI is

The desktop layer for agent operations.

RESI is for people who need to start work, understand what happened, and leave with something usable at the end.

Launch

Start work fast

Launch a run with the right pattern, target, and model from a single screen.

Observe

See what is happening

Follow progress, outputs, and topology as the system works.

Review

Review the results

Open files, inspect outputs, and package up the run when it is ready to hand off.

Core surfaces

Four views that cover the whole run.

Each view answers a different question: what to run, what is happening, how it is connected, and what came out.

Compose

Start the run.

Set the task, choose the shape of the run, and launch without digging through settings.

Direct controlEverything important is visible before you start.
Flexible runsUse the setup that fits the task instead of forcing one workflow.

Output

Watch progress live.

See status, messages, file updates, and results as they appear.

Topology HUD

See the system as a graph.

Understand which workers are active, how they connect, and where the run is getting stuck.

Live structureThe graph updates with the run instead of staying abstract.
Clear bottlenecksFind stalled or failed work faster.

Files

Open the artifacts.
